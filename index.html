Flexbox

-O modelo Flexbox é usado para organização.

-O conteúdo é colocado em uma coluna central verticalmente.

-Ele não ocupa toda a largura da tela, mas se ajusta ao conteúdo.

Grid Layout

-O modelo Grid Layout permite um maior controle sobre o posicionamento dos elementos.

-O conteúdo também está centralizado verticalmente.



RWD - Media Query

-O modelo Media Query utiliza uma regra para ajustar o layout em telas menores.

-A imagem é centralizada tanto horizontal quanto verticalmente na tela.

-As imagens são redimensionadas em telas menores (menos de 600px) para garantir que se ajustem à tela.
